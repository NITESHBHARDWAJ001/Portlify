# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-06

### Added
- Initial release of Portfolio Builder Platform
- Visual Canvas Editor with drag-and-drop functionality
- Component library with 9+ prebuilt blocks
- Template marketplace system
- README.md generator
- Portfolio publishing with custom URLs
- User authentication and authorization
- Responsive design with device preview modes
- Undo/redo functionality
- Zoom controls
- Real-time canvas editing
- MongoDB database integration
- RESTful API with Express.js
- JWT-based authentication
- Production-ready error handling
- Security middleware (Helmet, CORS)

### Components
- Hero Block
- About Block
- Project Card
- Skills Grid
- Timeline
- Markdown Block
- GitHub Stats
- Contact Form
- Typing Animation

### Features
- Create unlimited portfolios
- Save and load portfolios
- Publish portfolios publicly
- Create templates from portfolios
- Like and use community templates
- Export portfolios as README files
- Real-time canvas updates
- Responsive editing
- Global and component-level styling
